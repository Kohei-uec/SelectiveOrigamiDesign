Origami Face Maker

About

Origami Face Maker は折り紙の「顔」を設計するWebアプリケーションです。 自分好みの表情を作ってみてください。

How to Use

使いたかたは非常に単純です。
  1. 各パーツを選択します
  2. 紙の重なりを調整します（重なりの順序はパーツ同士の干渉があるため、その場でシミュレーションされます）
  3. 紙の色を調整します
  4. 折り紙上級者は「展開図折り」に挑戦！
また、他にも数多くの機能が備わっています。
  • Example から顔のサンプルを読み込めます
  • プロジェクトはブラウザまたはローカルに保存できます
  • 展開図は画像、SVG、CP形式で保存できます
  • 折り畳み図は画像、SVG、FOLD形式で保存できます
